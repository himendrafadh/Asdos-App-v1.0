# AsdosApp v1.0
Aplikasi Asisten Dosen (Asdos) adalah sebuah sistem yang didesain khusus untuk menyederhanakan dan mengoptimalkan tugas-tugas administratif dosen terkait data akademik mahasiswa.Fokus utamanya adalah pada pengelolaan nilai, yang meliputi proses penginputan,pembaruan, dan manajemen nilai secara keseluruhan. Aplikasi ini memiliki beberapa mode, yaitu :
1. Menu data mahasiswa
2. Menu data dosen
3. Menu mata kuliah
4. Nilai mahasiswa
